# 增强編輯器測試文章

這篇文章展示了新編輯器的全部功能。

## 拖放圖片

您可以直接將圖片拖放到編輯區域。

## 粘貼圖片

您可以直接從剪貼板粘貼圖片。

## 點擊插入圖片

您可以通過點擊按鈕選擇圖片。

![測試圖片](/src/assets/images/uploads/image-1765273656780-186176783.txt)

### Markdown格式支持

新編輯器支持完整的Markdown格式：

- **粗體文字**
- *斜體文字*
- `代碼塊`
- 代碼塊：
  ```javascript
  console.log("Hello World");
  ```
